# Ollama CV Creator Desktop

Local-first desktop app for creating professional CVs and cover letters with local Ollama models.

The app turns unstructured candidate context into a verified profile, uses optional job descriptions as tailoring context, and helps produce polished application documents without sending CV data to external AI services.

## Development

```bash
npm install
npm run dev
```

Desktop shell:

```bash
npm run dev:electron
```

## Checks

```bash
npm run typecheck
npm test -- --run
```
